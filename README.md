# 📊 Análisis y Procesamiento de Datos con Python (NumPy & Pandas)

Este proyecto realiza un flujo completo de **Data Wrangling** y análisis estadístico. Integra múltiples fuentes de datos, realiza limpieza profunda y genera reportes automáticos en formatos CSV y Excel.

## 🚀 Funcionalidades Principales
- **Integración Multifuente**: Combina datos generados con NumPy, archivos Excel locales y datos geográficos obtenidos mediante Web Scraping (Wikipedia).
- **Limpieza y Normalización**: Estandarización de textos (ciudades y regiones), manejo de valores nulos e imputación estadística (media y mediana).
- **Tratamiento de Outliers**: Aplicación de la técnica **Winsorizing** para mitigar el impacto de valores extremos sin perder datos.
- **Enriquecimiento de Datos**: Segmentación de clientes por "Etapa de Vida", detección de ventas en fin de semana y clasificación de tipos de compra.

## 🛠️ Stack Tecnológico
- **Python 3.x**
- **Pandas & NumPy**: Procesamiento y análisis.
- **Requests & BeautifulSoup (lxml)**: Extracción de datos web.
- **Openpyxl**: Generación de reportes en Excel con múltiples pestañas.
